# Voice to Cursor

📱 用手机语音/文字输入，实时同步到 Cursor/VS Code 的 AI 对话框。

## ✨ 功能特性

- 📝 **实时同步**：手机输入实时同步到电脑
- 🎤 **语音输入**：配合手机输入法的语音功能使用
- 🖼️ **图片支持**：支持选择相册图片或拍照发送
- 📋 **剪贴板互通**：可获取电脑剪贴板内容到手机
- 🔄 **替换当前行**：支持替换终端当前行内容
- 🤖 **AI 回复**：开启后 AI 回复会自动发送到手机
- 💬 **聊天记录**：手机端保存聊天历史，方便查看
- 🔗 **扫码连接**：通过二维码快速建立连接

## 🚀 快速开始

### 1. 安装手机 App

- 下载 [Expo Go](https://expo.dev/client)（iOS / Android）
- 克隆项目并启动手机 App：

```bash
git clone https://github.com/vonxq/voice-to-cursor.git
cd voice-to-cursor/mobile-app
npm install
npx expo start
```

- 用 Expo Go 扫描终端显示的二维码

### 2. 在 Cursor/VS Code 中启动

- 打开命令面板（`Cmd+Shift+P` 或 `Ctrl+Shift+P`）
- 搜索并运行 `Voice to Cursor: 启动服务`
- 会显示一个二维码面板

### 3. 手机扫码连接

- 在手机 App 中点击扫码按钮
- 扫描电脑上显示的二维码
- 连接成功后即可开始使用

## 📱 手机 App 功能

| 按钮 | 功能 |
|------|------|
| 📋 剪贴板 | 获取电脑剪贴板内容到手机 |
| 📥 当前行 | 获取终端当前行内容 |
| 🖼️ 相册 | 选择图片发送 |
| 📷 拍照 | 拍照发送 |
| 🗑️ 清空 | 清空输入框 |
| 📋 粘贴 | 仅粘贴内容到电脑 |
| 🔄 替换 | 替换终端当前行 |
| 🚀 发送 | 粘贴并回车发送 |
| 🤖 ON/OFF | AI 回复开关 |

## ⚙️ 配置说明

### 端口
默认端口 `9527`，确保该端口未被占用。

### 网络要求
- 手机和电脑必须在同一局域网
- 确保端口未被防火墙阻止

## 📝 命令

| 命令 | 说明 |
|------|------|
| `Voice to Cursor: 启动服务` | 启动 WebSocket 服务并显示二维码 |
| `Voice to Cursor: 停止服务` | 停止服务 |
| `Voice to Cursor: 发送回复到手机端` | 手动发送消息到手机 |
